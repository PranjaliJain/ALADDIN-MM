Commands to run:

1. If already done earlier, skip this step 
> cd /workspace/gem5-aladdin/src/aladdin/common
> make

2. Change directory to SHOC/mm
> cd /workspace/gem5-aladdin/src/aladdin/SHOC/mm

3. Make mm
> make run-trace -no-pie

4. If there are changes in mm.c or mm.h, run this
> g++ -O0 -fno-inline -o mm-instrumented full.s -lm -lz -lpthread -no-pie
> ./mm-instrumented

5. Run mm with config in example/ folder 
> cd example/
> /workspace/gem5-aladdin/src/aladdin/common/aladdin mm ../dynamic_trace.gz config_example
